# DAO Start Plan
Mycelium Protocol计划2026年1月开始，进行初始化启动，目标是通过出让5%左右的 GToken，开放认可协议的社区成员无许可（分两步建立pool）swap，从而获得部分资金，建立合法DAO组织，启动协议和社区的后续发展计划。

## 概述
Mycelium Protocol改进价值传递环节，连接价值创造者和价值消费者，提供社会资本新资产的识别和价值再分配。
基于以太坊，提供基建、协议和应用三个层面的公共物品。目前有一个AAStar开发组，完成了AirAccount和SuperPaymaster基建和COS72（OpenPNTs，OpenCards，OpenShops，OpenTasks等协议设计和部分开发）以及应用层的范例：Zu.Coffee。提供基于复杂系统科学的自组织、无许可的多样性选择，培育涌现土壤和协作网络，建设人类数字未来。

## DAO启动后治理

在完成DAO启动后，除了完成承诺的用途之外，还会独立进行DAO治理工作。
目前有三个核心topic：

### OpenNest机制建立
作为一个公共物品花园，GToken就是门票（还是可以退票的门票），而一个个的基建和协议，只是底层的土壤和管道，只有种植更多花朵，也就是应用，才能真实的提供数字福祉给社区成员。
### Burn/Minig参数初始化和管理
随着协议的服务展开，需要研究和分析，根据生态发展的速度和模式，调节burn的费用，加速或者限制协议生态的发展。
### GToken流动池
为社区成员提供基础的流动性服务，长期看，是保持协议生态价值、进入者数量和需求以及供给的平衡状态。
