## Access Model

The source code is licensed under Apache 2.0 — free to use, modify, and distribute.

Separately from the code license:

- **Protocol Network**: Accessing the live Mycelium Protocol network requires a
  GToken ticket. The code is open; the network is gated.
- **Organizations (>50 members)**: Apply for Network Access Authorization.
  See [COMMUNITY-USE-POLICY.md](https://github.com/MushroomDAO/launch/blob/main/docs/license/COMMUNITY-USE-POLICY.md).
- **Commercial use**: Code is free under Apache 2.0. For trademark use and
  network access for commercial purposes, contact jason@aastar.io.
