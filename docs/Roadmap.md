# Roadmap 2026/2027/2028
思维链条：个体-->自由和人性-->真菌协议协作-->经济模型探索-->社区+多样性，更多参考[人性之光小短文](./docs/HumanityLight.md)
社区是什么？所有不以资本和威权定义下价值观为目标的社会活动，涌现在秉持人性和自由为最终目标的组织（个体、小组、群体），都可以称之为社区。
社区最需要什么？Onbording+正反馈激励+Check=COS72(社区OS or ERP)+MySBT/MyNFT/MyPNTs+MyTask/Reputation
协议社区推出：Infra：合约+SDK+基础服务，Protocol：轻型协作DAO+服务协议孵化，DApp：应用模板+生命周期服务+经济模型
个体需要什么？：时间，空间，经济可持续，社区支持和社交诉求，释放自己的想象力，创造力和情感
