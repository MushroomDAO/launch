# 4Seas社区体系
四海感觉需要一个简单的会员体系，把历史居住过的人构成社区链接和社区网络并基于此做持续的运营；各种活动，也可以通过NFT和积分的形式，吸引新成员，具备常规性各地space活动，来持续积累成员参与感和归属感。
1. 会员体系：加入简单，基于Email，购买社区NFT（送一些天数），拥有会员身份
2. 积分容易：入住后积累积分，组织活动，参加活动，活动互动等社区提倡和鼓励的，给予积分。
3. 活动发布：活动的传统发布继续基于Lu.ma或者其他（web3events）等等，但活动报名的入口，除了在lu.ma放置会员RSPV的保留席位之外，还可以独立的页面发布Events，和社区关联，报名和活动组织等，需要基于此Events的预算。
4. 积分铸造和兑换：积分本质是一种记录和对未来的可持续，对于四海内部来说，需要有一个循环体系，让积分能够消耗掉形成循环，除了常规的例如兑换swag，兑换住宿等等常规手段外，还需要：
	1. 会员NFT发展为Hero NFT（需要积分兑换）
	2. 积分继续兑换好看的NFT 皮肤
	3. 积分在4Seas Web3小游戏内购买道具
	4. 兑换一些活动现场的食物、水和其他服务（CypherHouse的小卖部？）
5. 跨社区积分：这个体系开放给其他社区使用，大家一律平等去中心，但可以社区直接，进行积分兑换，积分类似于一个社区的流通和激励体系，不能滥发，不能失控，目前看，和服务的人数（会员数量）以及墒值（人员参与度和活动次数），类似货币银行学的简单的人数乘以人均GDP除以货币流通速度的货币发行逻辑。
6. 社区游戏：如上，开发比DarkForest好玩点的小游戏，不是复杂游戏
	1. 按泡泡：类似于抽奖，可以和其他游戏结合
	2. LBS游戏（ZK隐藏地址后的链上匹配，一次一周，但切换要付gas）
		1. 练习赛升级
		2. 组队别人带升级
		3. 积分和线下的Business，活动结合，日常兑换
			1. 例如学校食堂，7-11合作，CyperHouse合作等等
		4. 参加活动增加亲密度，组队升级更快
		5. NFT装甲和组件化的装备，积分兑换
		6. 游戏趣味性：捉迷藏等等
		7. 积分+NFT可以兑换限量手办
7. CypherHouse合作项目：
	1. 每个合作项目，单次有积分兑换上限
	2. 使用社区App扫码兑换：
		1. 信任交易：扫+付+当面交付
		2. 无信任交易：数字Item适合，例如NFT，需要中立合约担保和发货。
	3. 社区App是个联合App，任何社区都可以接入，发布活动，一键参加
	4. 社区积分显示和兑换，基础账户功能（归属哪几个社区，NFT等等）
8. Zuzalu体系协作
	1. 支持Zupass无缝登陆和使用
	2. 创建基于zupass的4Seas Pass，除了保密账户外，还有4Seas上面的NFT账户体系。
	3. 嵌入zuzalu.city,提供模块化能力（例如CypherHouse等等）
9. 社区成员创作和互动
	1. 创作NFT售卖
	2. 组队参加游戏
	3. 线下Meetup增加亲密度；
	4. 线下活动互动
