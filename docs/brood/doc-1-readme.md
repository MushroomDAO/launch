欢迎来到 BroodBrain，这是 Mycelium Protocol 的“协议孵化中枢”，我们的智脑系统。

## 开发团队（Dev Group）
1. **Web 看板**: Clone Brood repo到本地，运行：backlog browser命令，在浏览器打开 `http://localhost:6420`，点击 **Board**。
2. **终端查看**: 运行 `backlog task view 23 --plain`。
3. **添加修改**: 只有在本地界面才可以修改任务状态，添加任务等edit，对外发布互联网的是只读版本。

## 社区成员 (Community Member)
- **View Progress Report**: Check left side link
- **View Task Detail**: Click and view task items
