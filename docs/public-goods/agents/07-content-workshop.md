# AG-07 内容工坊 Agent

> **一句话**：给它你的产品/服务信息和目标平台，它帮你持续产出符合各平台调性的内容——让一个人运营多个账号变得可能。

---

## 目标用户

- 中小商家：微信公众号 / 小红书 / 抖音文案
- 独立创作者：个人品牌内容日历
- 初创公司：产品发布、活动宣传
- NGO / 社区：活动预告、报告宣传

## 核心痛点

| 痛点 | 现状 | Agent 解决后 |
|------|------|-------------|
| 内容更新跟不上 | 一个月发 2 条，账号沉默 | 给素材 → 批量生成一周内容 |
| 不同平台写法不同 | 小红书要 emoji，公众号要段落感 | 一键适配多平台格式 |
| 不知道写什么 | 脑子空白，对着屏幕发呆 | 提供话题建议 + 写初稿 |
| 文案质量参差 | 时好时坏 | 品牌声音档案 → 风格统一 |

---

## 核心概念：品牌声音档案（Brand Voice）

```yaml
# brand_voice.yaml
business_name: "小明奶茶"
persona: "活泼、亲切、有点小确幸感，像朋友聊天"
avoid: ["过于正式", "大词", "营销感太强"]
hashtag_style: true          # 小红书风格
emoji_density: medium        # 少量 emoji 点缀
call_to_action: "来店打卡"  # 默认行动召唤
audience: "18-30岁，城市白领，喜欢精致生活"
```

## 支持的内容类型

| 内容类型 | 平台 | 输入 | 输出 |
|---------|------|------|------|
| 产品推介 | 小红书 / 朋友圈 | 产品名 + 特点 | 800字种草文 + 标签 |
| 活动预告 | 微信公众号 | 活动时间/地点/主题 | 完整推文 + 标题备选 |
| 节日营销 | 抖音评论区 / 朋友圈 | 节日名 + 产品 | 3 条候选文案 |
| 用户故事 | 公众号 / 小红书 | 用户反馈截图 / 文字 | 整理成故事文章 |
| 每日一句 | 朋友圈 / 社群 | 主题词 | 一句走心文案 |
| 短视频脚本 | 抖音 / 视频号 | 产品 + 场景 | 30-60秒脚本 |

## 工作流程

```
用户提供素材（产品信息 / 话题 / 参考链接 / 用户反馈）
    │
    ▼
内容规划（Agent）
  → 根据品牌声音档案 + 目标平台
  → 建议本周内容日历（如果用户想）
    │
    ▼
草稿生成
  → 多版本备选（通常 2-3 个方向）
    │
    ▼
用户选择 / 修改意见
    │
    ▼
优化 + 平台适配
  → 小红书：加 # 标签、首段钩子句、结尾提问
  → 公众号：分段、加副标题、SEO 友好标题
  → 抖音：开头 3 秒钩子、口播节奏感
    │
    ▼
输出 + 可选同步到内容日历
```

## 内容日历模式

用户提供本月重点（节日 / 上新 / 促销），Agent 自动规划：

```
5月内容日历（小红书，目标：每周 3 条）

Week 1
  5/6 (一) 产品故事：我们的茶叶从哪里来
  5/8 (三) 节气营销：立夏特饮上线了！
  5/10 (五) 用户打卡合集

Week 2
  5/13 (一) 新品预热
  5/15 (三) 母亲节特别活动预告
  5/17 (五) 幕后故事：一杯奶茶是怎么做出来的
...
```

## 技术方案

```
品牌声音档案 + 用户输入
    │
    ▼
内容 Agent（Claude Sonnet）
  System prompt: 注入品牌声音档案
  Tools:
    - search_trending_topics()  # 热门话题检索（可选）
    - get_platform_rules()      # 各平台格式规范
    │
    ▼
多轮迭代优化
（用户反馈 → Agent 修改）
    │
    ▼
输出 Markdown / 复制到剪贴板 / 推送到内容日历工具
```
