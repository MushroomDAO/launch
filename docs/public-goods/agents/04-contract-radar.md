# AG-04 合同雷达 Agent

> **一句话**：上传合同，它帮你找出风险条款、解释法律术语、对比行业标准，让不懂法的人也能看懂自己签的是什么。

---

## 目标用户

- 自由职业者 / 独立顾问：签服务合同前的快速审阅
- 中小企业主：采购合同、合作协议、劳务合同
- 房客 / 购房者：租房合同、买卖合同
- 创业者：融资协议、股东协议初步理解（不替代律师）

## 核心痛点

| 痛点 | 现状 | Agent 解决后 |
|------|------|-------------|
| 合同看不懂 | 堆满法律术语 | 逐条通俗解释 |
| 不知道哪里有坑 | 要靠经验或找律师 | 自动标记风险条款 |
| 律师审合同贵 | ¥500-5000/份 | Agent 做初筛，律师只看关键点 |
| 多份合同对比 | 逐条手动比 | 上传两份 → 自动差异对比 |

## 功能清单（v1）

### 核心功能
- **风险扫描**：标记 5 类高风险条款（违约金/免责/排他/仲裁地/自动续签）
- **通俗解读**：每条转化为普通人能理解的语言
- **关键信息提取**：合同期限、金额、付款方式、交付条件、终止条件
- **问答模式**："这份合同里，如果我提前终止，要赔多少？"

### 辅助功能
- **双合同对比**：上传两版本 → 高亮差异
- **行业基准提示**：常见服务合同通常怎么约定，这份偏离了哪里

## 风险分级系统

```
🔴 高风险  — 可能造成重大损失或法律纠纷
🟡 中风险  — 需要注意但不致命
🟢 低风险  — 通常条款，了解即可
ℹ️  信息项  — 关键数据（金额/日期/名称）
```

**高风险示例**：
- "甲方可随时终止本合同，无需承担赔偿责任"→ 🔴 单方解约风险
- "乙方在合同期满后 2 年内不得从事同类业务"→ 🔴 竞业限制过宽
- "争议提交**甲方所在地**仲裁委员会仲裁"→ 🔴 仲裁地不利

## 输出格式示例

```markdown
## 合同审阅报告：《XX 服务合同》

### 基本信息
- 合同类型：技术服务合同
- 合同期限：2026-06-01 至 2027-05-31（12 个月）
- 合同金额：¥120,000（月付 ¥10,000）
- 付款方式：每月 5 日前支付上月款项

### 风险条款（共 3 项）

🔴 **第 8.2 条 — 违约金无上限**
> 原文："乙方逾期交付的，每逾期一日支付合同总额 1% 的违约金。"
> 解读：这意味着逾期 100 天就要赔付整个合同金额。行业惯例通常有违约金上限（如不超过合同总额的 20%）。
> 建议：协商加入"违约金总额不超过合同总额的 [X]%"上限条款。

🟡 **第 12.1 条 — 自动续签**
> 原文："合同期满前 30 日内，任何一方未书面提出终止，则合同自动续签一年。"
> 解读：如果你忘记提前 30 天发通知，合同自动再延一年。
> 建议：在日历上设置提醒（2027-05-01）。

🟢 **第 6.3 条 — 保密义务**
> 通常条款，双方互有保密义务，3 年期，合理。

### 关键提醒
- ⏰ 自动续签截止日期：**2027-05-01**（需在此前发终止通知）
- 💰 最大违约金风险：无上限（建议谈判修改）
```

## 技术方案

```
上传 PDF / Word
    │
    ▼
文档解析 + 结构识别
（条款编号识别、段落切分、表格提取）
    │
    ▼
两阶段分析
  阶段 1：结构化提取（LLM）
    → 合同类型、双方名称、金额、日期
  阶段 2：逐条风险评估（LLM + 规则引擎）
    → 按风险模式库匹配 + LLM 解读
    │
    ▼
Markdown 报告 + 高亮 PDF 输出
```

**风险模式库**（可扩展的规则集）：
```json
[
  {
    "pattern": "随时终止.*无需赔偿",
    "category": "单方解约",
    "risk": "HIGH",
    "advice": "建议加入解约赔偿条款"
  },
  {
    "pattern": "自动续签",
    "category": "自动续签",
    "risk": "MEDIUM",
    "advice": "设置日历提醒"
  }
]
```
