# AG-05 财务速记 Agent

> **一句话**：把票据、收支记录、银行流水扔给它，它帮你分类汇总、出月报、找异常——让没有财务的小团队也能看懂自己的钱去哪了。

---

## 目标用户

- 个体户 / 自由职业者：收入支出自动记录
- 10 人以下创业团队：无专职财务，创始人兼任
- 社区 / NGO / DAO：透明财务公示需求
- 电商卖家：订单收入 + 进货成本自动汇总

## 核心痛点

| 痛点 | 现状 | Agent 解决后 |
|------|------|-------------|
| 记账太麻烦 | 手动填表格，容易断 | 上传截图/流水 → 自动识别 + 分类 |
| 月底算不清楚 | 花了多少心里没数 | 自动月报：收入 / 支出 / 利润 |
| 不知道哪里花多了 | 要逐条看 | 自动按类别聚合 + 异常标记 |
| 报销凭证管理乱 | 纸质票据一堆 | 拍照 → OCR → 入账 → 归档 |

---

## 核心功能（v1）

### 输入方式
- **拍照上传票据**：OCR 识别金额 / 日期 / 商家 / 类别
- **上传银行流水 CSV**：工商行、招行、支付宝、微信导出格式直接支持
- **手动快速记录**："今天买了 3 袋咖啡 ¥87，算办公耗材"
- **导入电商订单**：淘宝/拼多多/抖音卖家后台 CSV

### 分类体系（可自定义）
```
收入
  └─ 销售收入 / 服务费 / 投资收益 / 其他

支出
  ├─ 人力（工资 / 外包）
  ├─ 办公（场地 / 耗材 / 设备）
  ├─ 营销（广告 / 礼品 / 活动）
  ├─ 运营（云服务 / 软件订阅 / 物流）
  └─ 其他
```

### 月报输出
```markdown
## 2026年4月 财务简报

### 收入：¥38,500
- 服务费收入：¥35,000（3 个客户）
- 其他：¥3,500

### 支出：¥22,100
- 人力外包：¥12,000（40%）
- 云服务：¥3,200（15%）
- 营销投放：¥4,500（20%）
- 其他：¥2,400

### 净利润：¥16,400（利润率 42.6%）

### ⚠️ 异常提醒
- 云服务支出环比上月增加 85%（¥1,730 → ¥3,200），建议核查
- 有 3 笔支出无票据，金额合计 ¥1,200
```

## 技术方案

```
票据图片 / CSV / 手动输入
    │
    ▼
解析层
  ├─ 图片：OCR（PaddleOCR 本地 / 腾讯 OCR API）
  ├─ CSV：规则解析器（按银行格式模板）
  └─ 手动：LLM 意图识别 → 结构化

    │
    ▼
分类层（LLM + 规则）
  → 自动归类 + 置信度
  → 低置信度 → 提示用户确认
    │
    ▼
存储层
  SQLite（本地单文件数据库）
  表：transactions, categories, attachments
    │
    ▼
报告层
  LLM 聚合 + 模板渲染
  → Markdown / Excel 导出
```

---

## 数据安全

- 本地 SQLite，无需上云
- 票据图片存本地文件系统
- 可选加密（SQLCipher）
- 仅 OCR 和 LLM 调用时发送文字内容，不发送原始图片到第三方（除非用户选云 OCR）

---

## DAO / 社区财务透明模式

特殊场景：DAO 需要对社区公示资金流向。

额外功能：
- 一键生成**公示版月报**（脱敏，隐藏供应商名称）
- 导出**Notion / 飞书**可嵌入格式
- 链上资金（Gnosis Safe）+ 链下资金合并视图
