# 数字公共物品发布计划 — Launch × Public Goods

> **定位**：Launch 不只是项目展示站，也是数字公共物品的发布平台。
> 每次发布都指向一个具体问题：让普通人和中小组织的数字生活更省力。
>
> **本仓库范围**：设计分析、需求定义、技术调研、产出文档。**不在此仓库开发代码**。
> 具体实现由独立子项目仓库承接，本仓库作为"公共物品图书馆"持续更新。

---

## 核心主张

**数字公共物品 = 任何人可用、不排他、能持续运行的数字工具或服务。**

我们发布的 Agent 工具满足以下条件：
- 解决**普通个体 / 中小组织**日常真实问题
- 能以极低成本（近乎免费）自部署或托管使用
- 数据留在用户侧，不强制上云
- 开源或源码可审计

---

## 发布原则

| 原则 | 说明 |
|------|------|
| **问题优先** | 先定义一个具体人群的具体痛点，再设计工具 |
| **最小可用** | 第一版只解决核心场景，不堆功能 |
| **可组合** | 每个 Agent 是独立模块，可与其他 Agent / 工具组合 |
| **渐进增强** | 从纯文本 RAG → 工具调用 → 多 Agent 协作，分阶段 |
