## D. 三年 Go-To-Market 节奏

### Phase 1（0–6 个月）：清迈实验室 — 物理验证

- **核心动作**：与清迈大学旅游学院联合开展课题，免费引入 10 家配合度高的种子商家。
- **驻店巡逻**：产品团队成员轮流驻店，获取一手反馈，迭代产品。
- **CAC**：≈ $100 / 商家（人工 + 合作费用分摊）。
- **基础设施**：从 35% 推进至 70%。
- **关键 KPI**：10+ 商家闭环跑通、第一笔链上结算、100+ 活跃表达者。

### Phase 2（6–18 个月）：城市合伙人计划 — 利益下放

- **核心动作**：从消费者中招募城市合伙人（City Partners），不是雇员，是协议节点。
- **激励机制**：分成制——每笔"超额订单"（超过基础免费配额，"大包裹另加票"）由协议自动给合伙人分润。
- **培训**：全套实训视频 + 数字公共物品搭建手册 + 首批商家落地洽谈实训。
- **目标城市**：清迈、大理、巴厘岛、里斯本、柏林等数字游民 + 小微商业聚集地。
- **关键 KPI**：11 个城市合伙人 / 城市、自动化匹配率 ≥ 30%。

### Phase 3（18–36 个月）：AI 自动化 — 无许可加入

- **核心动作**：AI Agent 自发现成熟（进度 20% → 100%），系统进入"自动驾驶"。
- **入驻方式**：全球任何商家无许可、零门槛接入。
- **培训**：纯线上视频 + AI 助手引导。
- **关键 KPI**：年营收 $10M、盈亏平衡、40% 反哺协议、自增长正循环。

## F. 风险与抗风险设计（投资人压力测试）

| 风险 | 抗风险设计 |
|------|-----------|
| **冷启动陷阱** | CMU 学术合作 + 免费 + 驻店——把信任门槛降到最低 |
| **Web2 平台抄袭** | 平台没法抄"40% 反哺"和"开源底层"；抄了就变成开源生态的一部分 |
| **Token 价值捕获悖论** | 三层收入流（基础+AI+Pro）+ ARPU $150/月 + 11 社区 = $10M 盈亏平衡——Token 价值有真实商业现金流支撑 |
| **作弊 / Sybil Attack** | 分润触发条件是"真实成交"——欺诈方骗的是商家自己的预付积分，协议无系统性资金池被薅风险 |
| **商业公司利益冲突** | 物理约束（不可改合约）+ 经济诱因（20% token + 唯一授权）+ 社区主权（三年后可投票引入竞争） |
| **跨行结盟汇率波动** | 借鉴航空联盟里程互换——经济模型稳定、汇率不会频繁波动、合规可审计 |
| **退出路径** | Token 升值 + 股权分红 + 高价值模块（如 Sin90 / iDoris）独立拆分上市可能 |
