## B. 在做什么——Mycelium Protocol

### B.1 一句话定位
**"数字公共物品的基础设施 + 协议 + 网络"**——为普通个体在 AI 时代保有数字主权、获得数字福祉提供开源协作系统。

### B.2 为什么是"现在"——意义经济的逻辑闭环

来自 blog《Rise of Meaning Economy》：

1. AI 已成为"独立完成生产闭环的替代性生产力"
2. 资本意志的唯一目标是自我增殖，会用 AI 替代每一个可替代的人类角色
3. 低生育率是社会有机体对"人沦为资本耗材"的负反馈
4. 不可被 AI 替代的内核：**想象力、创造力、情感**
5. 由此诞生三类新角色：**表达者、创造者、建设者**
6. Nayebi (2025)：AI 达到当前自动化生产力的 5–7x 即可支撑 11% GDP 的 UBI；快速进展情景下 **2028 年** 即可跨越——窗口紧迫

→ 推论：必须在 2028 之前为这些角色建好"基础设施 + 协议 + 网络"

### B.6 商业公司的三个收入来源（Jason 自述）

1. **Token 升值**——21M 限量 + 公共品越来越多 → 门票需求增长
2. **Pro 版服务**——开源免费版 + 商业 Pro 版（参考 Red Hat / Linux + 红帽 类比）
3. **去中心化应用网络上的创新**——具体首个商业化落地是**商业积分 / 客户忠诚度积分**
   - 三方：中小商家 / 消费者 / 表达者（消费者写体验内容）
   - 通过区块链构建"消费 → 表达 → 传播 → 兑换"的闭环
