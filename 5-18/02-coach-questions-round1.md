# 02 — 教练第一轮问题

> **教练立场**：我读完你的口述、launch.mushroom.cv、jhfnetboy.pages.dev、两篇 blog。
> 你的故事在"理想"这一头很完整、很有感染力（数字公共物品、意义经济、AI 与资本的对抗）。
> 但在"路演"这一头有几个洞——杭州那一周，一个见你 5–10 分钟的 MBA 同学/前同事/陌生投资人，需要在这几个洞里听到答案，否则会礼貌性地点头然后再不联系。
>
> 我会分四组共 12 个问题。**每组先回答完再进入下一组**——避免一次性面对全部，写不动。
> 回答方式：直接在每个问题下面用一段话回答（不用结构化），有不确定的地方就写"我不确定，倾向是 X"——比给一个虚假的答案更有用。

---

## 第 1 组：受众认知（最关键，决定话术怎么剪裁）

### Q1.1 — 杭州这一周你大概会见多少人？请大致分一下类。
比如："5 个 MBA 同学（其中 2 个在金融、3 个在传统行业），3 个前同事/前领导（都是技术管理者），1–2 场 demo day 上不认识的投资人，可能还有 1–2 个老朋友。"
我需要这个数字感来决定话术变体投入多少。

### Q1.2 — 在这些人里，谁**最有可能**真的拿出 5–50 万人民币（或 5 千–5 万美元）？
不是"我希望谁会"，是"我冷静评估，谁的画像最匹配"。如果你只能锁定 1–2 个具体的人，提前定制话术更有效。

### Q1.3 — 在这些人里，谁如果对项目"不看好"，你会**真的难受**？
（不是"礼貌性失望"，是"靠，他没看懂或者他根本不认我"那种感受。）
这一题是为了找出**情感 stake** 最高的对象——他们决定你 demo day 之前会不会自我审查、缩手缩脚。

## 第 2 组：商业模式的硬骨头（投资人的 90% 提问会落在这里）

### Q2.1 — "20% token 给商业公司、商业公司 40% 收入回开源"——这个协定**写在哪**？
你说"这是写死的协定"。是写在：
- (a) Solidity 合约里（如果是，请贴 contract 路径或 GitHub link）
- (b) MushroomDAO 章程 / 多签控制
- (c) 还在心里 / 文档草案 / 未上链
?
这个问题决定投资人愿不愿意付钱——如果只是 (c)，5 年后协议被你或继任者改掉的成本是 0，那他买的不是协议，是对你这个人的信任。这是**完全不同的两种交易**。

### Q2.2 — GToken 价值的"门票升值"逻辑能再具体一点吗？
你的论证是：21M 限量 + 公共品越来越多 → 需求增长 → token 升值。
但门票（一次性）和 token（永久持有）是两种东西。请回答：
- 用户**必须持有 GToken** 才能做哪个具体动作？（治理之外）
- 有没有"使用即销毁 / 锁仓"的机制？
- 如果没有，10 年后流通盘会变成什么样——巨鲸囤积？
我不是在挑刺，是在帮你把这一段故事变得**经得住技术型投资人 30 秒的追问**。

### Q2.3 — "商业积分 / 客户忠诚度积分"——这是 3 年内最现实的现金流吗？为什么？
你提到三个收入来源：token 升值 / Pro 服务 / 去中心化应用创新（首个落地是商业积分）。
请挑**一个**作为 3 年内的"现金流主引擎"，并回答：
- 为什么是它，不是另外两个？
- 第一个真实付费客户大概长什么样？（行业、规模、付多少）
- 他**为什么用你**而不用 Stripe / 微信会员 / Loyalty SaaS？
路演时如果你六个产品并排讲，听众一个都记不住。**挑一个讲深，比六个讲浅有效 10 倍**。

### Q2.4 — 这是公益还是生意？
（这是 MBA 同学一定会用某种方式问的话——"你这个能赚钱吗"。）
你目前的语言混合了"数字公共物品（公益）"和"商业公司 + token 升值（生意）"。请你**自己**用一句话定义你的项目位于光谱的哪个点：
- (a) 公益项目，3 年靠捐赠 / Grant 维持
- (b) 双轨：开源协议 + 商业公司，长期看商业公司能 break-even
- (c) 商业公司是核心，开源协议是产品的"开放外壳"
- (d) 其他

不同位置面对不同投资人——(a) 找基金会，(b) 找有耐心的天使，(c) 找 VC——**你不能假装三个都做**。

## 第 3 组：投资回报的具象化（投资人最关心）

### Q3.1 — 5 万美元的人，3 年后能拿到什么？
请用**具体数字**回答下面三个组合中的一种或几种，不要泛泛地说"token + 治理权 + 道义回报"：
- (a) X 枚 GToken（按当前 $0.15 = 33 万枚）；3 年后 token 估值情景：保守 / 中位 / 乐观
- (b) 商业公司股权 X%
- (c) 收入分成 X%（哪个产品的 / 多久 / 上限）
- (d) 都不是——这是**纯捐赠**，写在 PR 上、不期待财务回报

如果是 (a)，请给一个**三档情景** + 你认为情景概率（粗略 30/50/20 即可）。
**没有数字的回报承诺，专业投资人是不会签字的**——哪怕是"这是捐赠，没有金额回报"也是一个清晰的答案。

### Q3.2 — 失败情景：3 年后如果项目没做起来，钱哪去了？
（你要主动准备这个答案——投资人不会问，但他们脑子里在问。）
请说清楚：
- 钱用在哪些**具体支出**上（人头、AI 订阅、服务器、论文费）
- 是否有**最坏情况下的剩余资产**（代码开源、用户数据、社区资源）能让他得到一些"残值"
- 你**个人**是否承担任何责任（道义、声誉、还款承诺）

### Q3.3 — Why now & why you——30 秒回答。
*Why now*：为什么 2026 年这件事一定要做？（不是"AI 会替代人类"那种远景，而是"今年不做，6 个月后就晚了"）
*Why you*：为什么你这个人（vs. 任意一个 web3 + AI 创业者）能做成？
