GToken · Governance Token

协议参与凭证
Protocol Participation Token

GToken 是 Mycelium 协议的治理与访问凭证。持有即参与，质押即解锁。

Core Functions

GToken 能做什么 / What GToken Does

治理投票
Governance Voting
  • 持有 GToken → 有权对协议提案投票
  • 包括：协议参数、资金使用、新功能优先级
  • 链上可查，链上执行
质押 · 注册角色
Stake & Register Role
  • 质押 GToken → 在 Registry 合约注册协议角色
  • Community：创建社区节点，获得 ENS 子域名
  • Individual：建立链上数字身份
  • Operator：运营 SuperPaymaster gas 支付服务
协议访问权
Protocol Access
  • join 协议：消耗 GToken（burn-on-join）
  • exit 协议：消耗 GToken（burn-on-exit）
  • 销毁机制 + 21M 硬顶 = 自然稀缺性
