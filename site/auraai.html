MISSION

为什么是 AuraAI

AI 正在替代人类的生产力角色，但有三件事 AI 永远替代不了—— 想象力 (Imagination)、创造力 (Innovation)、 情感 (Emotion)。这是意义经济 (Meaning Economy) 的核心命题：让 AI 处理生产，让人类专注意义。

为此，AuraAI 围绕三类新角色—— 表达者 / Expressor、 创造者 / Creator、 建设者 / Builder ——构建 AI Native 基础设施。每一个 AuraAI 组件都是 本地优先 / 隐私安全 / 开源可审计 的—— 你的数据不卖给训练公司，你的 Agent 不替别人监控你。

核心组件 / Core Components

Apache 2.0 license · Self-hostable · Open weights philosophy（基于 OSS LLM 微调，不绑大厂 API）

iDoris

Beta

社区大脑 / Community Brain——基于"协议 + 联盟"训练的多层 AI 模型， 为社区提供：

  • 消费建议（基于积分网络中的真实表达）
  • 招聘 / 评测 / 合作机会智能匹配
  • 投资决策辅助（社会资本评估模型）
  • 跨社区资源调度

核心创新：联邦分层权重训练，每个社区都有自己的私有微调层，但能共享底层知识池。

iDoris · Apache 2.0

Agent24

Beta

通信 + 协议 Agent——24/7 持续运行的智能体，处理跨场景协作：

  • 跨 IM 平台消息聚合（微信、Telegram、Discord、邮箱）
  • 自动安排会议 / 协调 / 任务派发
  • 协议层签名（自动签订商业积分跨店结盟合同）
  • 超额带宽 = 商业公司收入来源（基础免费，Pro 计费）
Agent24 · Apache 2.0

OpenCrab

Research

分布式数据爬取 + 整理协作框架——为 iDoris 等下游 AI 提供高质量数据集。 社区贡献者可"Operate" OpenCrab 节点赚 PNTs。

OpenCrab · Apache 2.0

Sin90 (个人 OS)

Beta

个人 AI 操作系统——本地优先的 Agent 团队 + 知识管理 + 隐私数据。 运行在用户自己机器上，数据永不上传。可挂 OpenAI/Claude API 或纯本地 LLM (llama.cpp / Ollama)。

Sin90 · Apache 2.0

愿景 / Vision
