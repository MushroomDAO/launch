## 项目描述草稿（≤200字）

```
Mycelium Protocol 是一个区块链与 AI 融合的 Agent 协作网络基础设施。

核心创新：通过可定义的协作协议（Collaboration Protocol）和协作流程，
让多个 AI Agent 能够以去中心化、可验证的方式协同完成复杂任务。

技术基础：
- 账户抽象（ERC-4337/AirAccount）提供无 Gas 操作体验
- SuperPaymaster 处理 Agent 之间的链上结算
- Registry 合约管理 Agent 角色和权限
- 链上协作协议确保行为可追溯、可审计

应用场景：
- 中小企业数字化：微信客服、研究助手、图片生成等 Agent 工具
- DAO 治理自动化：提案分析、投票统计、链上执行

项目由浙大校友 Jason Jiao 主导，团队来自 AAStar 开源社区。
```

字数：约170字 ✅
