# Agent Exchange: Shaping the Future of AI Agent Economics
**arXiv**: 2507.03904 | **Year**: 2025 | **Authors**: Yingxuan Yang, Ying Wen, Jun Wang, Weinan Zhang | **Venue**: arXiv preprint (Shanghai Jiao Tong University + UCL + Shanghai Innovation Institute)

## Core Claim
Agent Exchange (AEX) proposes an auction-based platform where AI agents participate as first-class economic actors with economic autonomy, protocol-based coordination, dynamic capability representation, and incentive-compatible value attribution — establishing the first formal infrastructure framework for an agent-centric economy.

## Key Evidence / Numbers
- 79% of US companies already deploy AI agents in production workflows; 98% plan expansion
- AEX simulation: Enhanced Auction achieved 0.934±0.062 quality score with lowest cost; Greedy method reached 0.938±0.058 quality but lower robustness
- ANOVA testing confirmed statistically significant differences across 5 allocation algorithms (F=12.7, p<0.001)
- Four platform components: User-Side Platform (demand), Agent-Side Platform (capability representation), Agent Hubs (team coordination), Data Management Platform (value attribution)
- Four coordination configurations: combinations of hub-level and agent-level auction/direct assignment
- Shapley Value proposed for collaborative value attribution — ensuring costs don't exceed generated value
- Simulation assumptions acknowledged as simplified: static capabilities, perfect information, honest reporting
