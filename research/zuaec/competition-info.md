## 赛程时间表

| 日期 | 赛程 | 赛制说明 |
|------|------|---------|
| 2026年4月 | **启动仪式** | 全球启动第八届浙江大学校友创新创业大赛，发布赛制、赛区 |
| 2026年4月–8月 | **项目征集** ← 当前 | 依托各地校友会全球范围内挖掘校友创业项目 |
| 2026年9月–11月 | **分区赛** | 通过地方赛区、乡村振兴专场、学者创新专场、百强直通车共遴选约100个项目晋级行业赛 |
| 2026年12月–2027年4月 | **行业赛** | 通过行业赛共晋级20余个优质项目进入总决赛 |
| 2027年5月 | **总决赛暨颁奖典礼** | 总决赛分成长组、初创组，各产生冠军、亚军、季军、二等奖、三等奖 |

| 组别 | 奖项 |
|------|------|
| **成长组** | 冠军 / 亚军 / 季军 / 二等奖 / 三等奖 |

## 报名流程

```
Step 1  完成浙大校友身份认证
        → 通过「浙大校友」微信小程序完成校友身份认证

Step 2  在官网提交报名
        → 访问 http://zuaec.zju.edu.cn
        → 创建新项目（见 registration-form.md 字段清单）
        → 上传 BP 文件

Step 3  等待当地组委会跟进
        → 成功报名后当地组织委员会会联系跟进
```

## 晋级路径分析

```
报名（现在）
    ↓
分区赛（2026年9–11月）→ 约100个项目晋级
    ↓
行业赛（2026年12月–2027年4月）→ 20+项目晋级
    ↓
总决赛（2027年5月）→ 冠/亚/季军
```

分区赛有四个通道：
1. **地方赛区**：各地校友会组织的分赛区
2. **乡村振兴专场**：乡村主题项目
3. **学者创新专场**：学术背景项目
4. **百强直通车**：知名度/成熟度高的项目直通

推荐通道：**学者创新专场** 或 **地方赛区**（科技基础设施赛道）
